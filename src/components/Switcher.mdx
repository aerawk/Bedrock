# Switcher

## Use case

The SplitSwitcher component is designed to create a split layout based on fraction, but will switch to a Stack layout at a switchAt size (default is the smallOnly breakpoint).
The SplitSwitcher has the same API as the Split Component, plus a `switchAt` prop. when the Switcher is less than or equal to that pixel width, it will switch to a `Stack` component.

## How to install

If you haven't already, install `styled-components`:

```bash
  npm install styled-components
```

Then install the stack component

```bash
npm install @bedrock-layout/switcher

import { SplitSwitcher } from '@bedrock-layout/switcher';
```

## Properties

It has the same API has the [`Split` component](/src-components-split) plus:

| Property | Type   | Default              |
| -------- | ------ | -------------------- |
| switchAt | number | smallOnly breakPoint |

## IE11 Fallback

For browsers that do no support CSS Grid, it will fall back to a flex column.
